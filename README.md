# นับสต๊อคสินค้า (Stock Count App)

เว็บแอปฟอร์มนับสต๊อคเบียร์/เครื่องดื่ม แทนกระดาษ ใช้งานง่าย ไม่ต้องมี server หรือฐานข้อมูล
ข้อมูลถูกเก็บไว้ในเบราว์เซอร์ของเครื่องที่กรอก (localStorage) เมื่อกรอกเสร็จให้กดโหลดเป็นรูปภาพแล้วส่งเข้ากลุ่ม LINE

## วิธีใช้งาน

1. เปิดเว็บ (ลิงก์ GitHub Pages หรือเปิดไฟล์ `index.html`)
2. กรอก/แก้ไขจำนวนคงเหลือ, จำนวนใน POS, % คงเหลือของแต่ละ Tap, รายการของทอดที่ต้องสั่งเพิ่ม, Shopping list
3. เพิ่ม/ลบรายการสินค้าได้ด้วยปุ่ม "+ เพิ่มสินค้า" หรือปุ่ม ✕ ท้ายแถว
4. กด **"📷 โหลดเป็นรูปภาพ"** เพื่อดาวน์โหลดไฟล์ PNG ของฟอร์มที่กรอกแล้ว แล้วนำไปส่งในกลุ่ม LINE
5. เมื่อจะเริ่มนับรอบถัดไป กด **"🔄 เริ่มรอบนับใหม่ (ล้างจำนวน)"** เพื่อล้างเฉพาะตัวเลข (ชื่อสินค้ายังอยู่ครบ)
6. หน้าเว็บจะขึ้นแบนเนอร์เตือนอัตโนมัติในวันที่ 5, 10, 15, 20, 25, 30 ของทุกเดือน (รอบนับสต๊อค)

**หมายเหตุ:** ข้อมูลอยู่ในเบราว์เซอร์ของเครื่องนั้นๆ เท่านั้น ถ้าเปลี่ยนเครื่อง/เปลี่ยนเบราว์เซอร์ ข้อมูลจะไม่ตามไปด้วย
(ประวัติการนับแต่ละรอบให้ดูจากรูปที่ส่งไว้ในกลุ่ม LINE)

## เปิดใช้งานบนเครื่องตัวเอง (ทดสอบ)

ไม่ต้องติดตั้งอะไรเพิ่ม เปิดไฟล์ `index.html` ด้วยเบราว์เซอร์ได้เลย หรือถ้าต้องการรันผ่าน local server:

```bash
python3 -m http.server 8000
```

แล้วเปิด http://localhost:8000

## Deploy ขึ้น GitHub Pages

1. สร้าง repository ใหม่บน GitHub แล้ว push โค้ดนี้ขึ้นไป
2. ไปที่ **Settings → Pages**
3. เลือก Source: `Deploy from a branch`, Branch: `main` (หรือ branch หลักที่ใช้), Folder: `/root`
4. กด Save รอสักครู่จะได้ลิงก์ เช่น `https://<username>.github.io/<repo>/`
5. แชร์ลิงก์นี้ให้พนักงานเปิดใช้งานผ่านมือถือ/คอมได้เลย
